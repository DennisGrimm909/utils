Want to display text with an overline in the browser? It's a but tricky as font support for Unicode Character \u0305 Combining Overline is patchy. However Arial works well and is available by default in recent Android, Windows, macOS, iOS (not sure which versions)

macOS: Lucida Grande

Android: Arial (needs to come before Helvetica as Android also has that - not working)

iOS: Arial (needs to come after Lucida Sans Unicode as Windows also has Helvetica)

Here's a test with <span style="text-decoration:overline">here the text has overline,</span> 0.1234

So here's the magic CSS code:

<p style="Arial, Sans-Serif">
