Want to display text with an overline in the browser? It's a but tricky as (a) font support for Unicode Character \u0305 Combining Overline is patchy and (b) good support is in different fonts on each platform

Windows: Lucida Sans Unicode

macOS: Lucida Grande

Android: Arial (needs to come before Helvetica as Android also has that - not working)

iOS: Helvetica (needs to come after Lucida Sans Unicode as Windows also has Helvetica)

Here's a test with <span style="text-decoration:overline">here the text has overline,</span> 0.1234

So here's the magic CSS code:

<p style="Lucida Sans Unicode, Lucida Grande, Arial, Helvetica, Sans-Serif">
